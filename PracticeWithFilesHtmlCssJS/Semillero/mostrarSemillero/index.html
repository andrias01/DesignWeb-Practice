CSS

CSS (Cascading Style Sheets) es un lenguaje de programación que se utiliza para definir la apariencia de páginas web. También se le conoce como Hojas de Estilo en Cascada.

Aprenderemos los conceptos básicos de CSS.

CSS es el lenguaje de estilos que define la apariencia de las páginas web.

Podemos cambiar colores, tamaños y posiciones con CSS.
